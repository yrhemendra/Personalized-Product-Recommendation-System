# Personalized Product Recommendation System

A machine learning project that recommends personalized products using collaborative filtering and matrix factorization techniques.

---

## Overview

Recommendation systems are one of the core technologies behind platforms like Amazon, Netflix, Spotify, and Flipkart.

This project builds an end-to-end recommendation engine capable of generating personalized product suggestions using historical user-product interaction data.

---

## Features

✔ User-Based Collaborative Filtering

✔ Item-Based Collaborative Filtering

✔ Matrix Factorization (SVD)

✔ Similar Product Recommendation

✔ Personalized Top-N Recommendations

✔ Streamlit Web Interface
